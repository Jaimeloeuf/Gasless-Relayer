Signs a incoming transaction with private key to pay for user's Gas.
Example
const Tx = { to: ..., ... }
const signedTx = fetch('/relayer', {method: POST}).send(Tx)
// {success: true}
Parameters:
Name Type Description
path string relative express path
middleware callback express middleware
scw_address Address of the user's own Smart Contract Wallet
to The address to make the transaction to from the wallet
value The value to sent over from the user's smart contract wallet to the
data The data to be sent over to the contract the wallet is interacting with
txHash The hash of the original transaction to be signed over
sigs The signature(s) of the txHash which will be sent over to the wallet for verification
Returns:
Signed transaction is returned to the user

object
